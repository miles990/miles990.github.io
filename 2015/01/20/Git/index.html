Git 學習筆記

Git

參考: http://git-scm.com/book/zh-tw/v1/

Git 有三種檔案狀態的表達方式

  • committed(已提交)
  • modified(已修改)
  • staged(暫存)

常用指令

  • 初始化 git 儲存庫: 切換到欲使用版本控管的目錄下，執行以下命令
    $git init
    此動作會在目錄下建立名為 .git 的子資料夾，此資料夾下存放 git 相關的記錄檔案

  • 將目標 <檔案> or <目錄> 加到準備提交的狀態: (準備commit，尚未送出)
    $git add <檔案> or $git add <目錄>

  • 複製現有 <儲存庫(reposity)>
    $git clone <repo>

  • 檢視檔案的狀態
    $git status
